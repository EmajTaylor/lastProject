Facts bout HTML and CSS

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It provides the basic building blocks for web pages, such as headings, paragraphs, links, and images. CSS (Cascading Style Sheets) is used to style and layout HTML elements. It controls the visual presentation of web pages, including colors, fonts, and spacing.
